# Maze Solver

## About the Project
This project is a fun and educational maze-solving application.  
It can solve both the built-in maps and any custom maps created by the user.

---

## Features
- Includes 4 default maps to help users understand how the program works
- Ability to create custom maps
- Ability to save created maps and use them later
- The goal of the program is not to find the shortest path, but to eventually reach the exit, even if it needs to check every possible path

---

## How to Use

### Running the Program
1. Open the `.exe` file
2. If the program does not open in fullscreen mode, switch it to fullscreen manually

---

### Using Default Maps
- Choose one of the options from `1` to `4`

---

### Creating a Custom Map
- Select option `5`

#### Step 1 — Select Map Size
First, choose the height and width of the map.

#### Step 2 — Select Start Point
Choose the starting point.

#### Step 3 — Select End Point
Choose the ending point.

- Use the `Arrow Keys` to move around the map
- Press `Space` to select the desired location for the start and end points

> Note: Corner cells cannot be selected as the start or end point.

---

### Creating Paths
Now you need to create the maze path:

- Use the `Arrow Keys` to create paths  
  Every cell you move through becomes part of the path.

- Use the `W`, `A`, `S`, `D` keys to remove paths  
  Every cell you move through becomes a wall again.

- Press `Space` to finish building the map.

---

### Saving the Map
After creating the map, the program will ask whether you want to save it or not.

Finally, the program will solve the maze and display the discovered path.

---

### Using Saved Maps
- If you already have saved maps, select option `6`

# حل‌کننده مارپیچ

## درباره پروژه
این پروژه یک برنامه سرگرمی و آموزشی برای حل مارپیچ است.  
این برنامه می‌تواند هم مپ‌های پیش‌فرض و هم مپ‌هایی که توسط کاربر ساخته می‌شوند را حل کند..

---

## قابلیت‌ها
- دارای 4 مپ پیش‌فرض برای آشنایی با نحوه عملکرد برنامه
- امکان ساخت مپ‌های سفارشی
- امکان ذخیره کردن مپ‌ها و استفاده مجدد از آن‌ها
- هدف برنامه پیدا کردن سریع‌ترین مسیر نیست؛ بلکه رسیدن به خروجی است، حتی اگر نیاز باشد تمام مسیرها بررسی شوند

---

## نحوه استفاده

### اجرای برنامه
1. فایل `.exe` را اجرا کنید
2. اگر برنامه به صورت فول‌اسکرین اجرا نشد، به صورت دستی آن را فول‌اسکرین کنید

---

### استفاده از مپ‌های پیش‌فرض
- یکی از گزینه‌های `1` تا `4` را انتخاب کنید

---

### ساخت مپ سفارشی
- گزینه `5` را انتخاب کنید

#### مرحله 1 — انتخاب اندازه مپ
ابتدا ارتفاع و عرض مپ را مشخص کنید.

#### مرحله 2 — انتخاب نقطه شروع
نقطه شروع را انتخاب کنید.

#### مرحله 3 — انتخاب نقطه پایان
نقطه پایان را انتخاب کنید.

- با استفاده از کلیدهای `Arrow Keys` می‌توانید روی مپ حرکت کنید
- با کلید `Space` مکان موردنظر را برای نقطه شروع و پایان انتخاب کنید

> نکته: خانه‌های گوشه‌ای قابل انتخاب به عنوان نقطه شروع یا پایان نیستند.

---

### ساخت مسیرها
حالا باید مسیرهای مارپیچ را ایجاد کنید:

- با استفاده از `Arrow Keys` مسیر ایجاد کنید  
  هر خانه‌ای که از روی آن عبور کنید به عنوان مسیر ثبت می‌شود.

- با استفاده از کلیدهای `W`, `A`, `S`, `D` مسیرها را حذف کنید  
  هر خانه‌ای که از روی آن عبور کنید دوباره به دیوار تبدیل می‌شود.

- با کلید `Space` ساخت مپ را نهایی کنید.

---

### ذخیره مپ
بعد از ساخت مپ، برنامه از شما می‌پرسد که آیا می‌خواهید آن را ذخیره کنید یا خیر.

در نهایت برنامه مپ را حل کرده و مسیر پیدا شده را نمایش می‌دهد.

---

### استفاده از مپ‌های ذخیره‌شده
- در صورت وجود داشتن مپ ذخیره‌شده، گزینه `6` را انتخاب کنید
